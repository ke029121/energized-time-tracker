## 📝 Using energized-time-tracker

### Starting Your First Task

- Open the app, and you will see a simple window with a list of tasks or no tasks if this is your first time.  
- Click the “New Task” button to add a task name. Use something easy to remember like "Work", "Study", or "Exercise".
- To start tracking time, select the task and press the START button or use the global hotkey (default hotkey will be shown in the app settings).
- The timer will run until you stop or pause it.

### Pausing and Stopping

- Click PAUSE if you take a break but want to continue timing later.  
- Click STOP when you finish the task or want to switch. This saves your recorded time for that session.

### Away From Keyboard (AFK) Detection

- The app automatically stops tracking when it detects no keyboard or mouse activity for a while. This prevents wrongly counting idle time.

### Viewing Your Statistics

- Switch to the "Stats" tab to see charts of your time spent on tasks.  
- You can view daily or weekly summaries to understand where your time goes.  
- Use filters to examine specific dates or tasks.
