# Energized Time Tracker

Простой и мощный десктопный тайм-трекер для учета времени, потраченного на различные задачи.

[English version (Английская версия)](README.md)

## Функционал

### 1. Учет времени
- **Запуск/остановка таймера:** Легко начинайте и останавливайте отсчет времени для любой задачи.
- **Описание задачи:** Добавляйте заметки к каждой сессии, чтобы знать, над чем вы работали.
- **Авто-именование:** Если вы начинаете новую сессию с тем же названием, к нему автоматически добавится номер (например, "Работа (2)").

### 2. Глобальное управление
- **Глобальный хоткей:** Запускайте и останавливайте таймер из любого места в системе с помощью комбинации клавиш `Ctrl+Shift+T`.
- **Системный трей:** Приложение сворачивается в трей, откуда можно управлять таймером, открывать окно или закрывать программу. Иконка в трее меняет цвет в зависимости от того, запущен ли таймер.
- **Всплывающие уведомления:** Получайте ненавязчивые уведомления о запуске и остановке таймера.

### 3. Отслеживание активности
- **AFK-трекер (Away From Keyboard):** Таймер может автоматически останавливаться, если вы неактивны за компьютером (не двигаете мышью и не нажимаете клавиши). Время простоя настраивается.

### 4. История и редактирование
- **Просмотр истории:** Все сессии сохраняются и группируются по дням. Вы можете просматривать историю за любой день.
- **Редактирование записей:** В истории можно изменять время начала, окончания и описание для любой сессии.
- **Удаление записей:** Ненужные сессии можно легко удалить.

### 5. Статистика
- **Визуализация данных:** Просматривайте статистику по затраченному времени на разные проекты в виде наглядных диаграмм.
- **Фильтрация по времени:** Анализируйте данные за сегодня, последние 7 дней или за все время.

### 6. Удобство использования
- **Автодополнение:** При вводе названия задачи приложение предлагает варианты из уже существующих проектов.
- **Сохранение данных:** Все данные хранятся локально в формате TOML в папке `%LOCALAPPDATA%/TimeTracker`.
- **Современный интерфейс:** Темная тема оформления на базе FlatLaf.
